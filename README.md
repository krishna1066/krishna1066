### APPROACH

My work philosophy centers on three principles:

**Accessible First** — Every project should run in a browser without installation barriers. WebAssembly, WebGL, and modern JavaScript frameworks enable sophisticated experiences that once required native apps.

**Production Ready** — Demos are interesting. Shipped products are valuable. I focus on complete, deployable solutions with real-world utility.

**Human-Centered** — Technology should adapt to users, not the other way around. This drives my work in assistive AR and accessible computer vision applications.
